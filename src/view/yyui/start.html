开始

简要的介绍 YY UI，以及下载、使用。

下载

YY UI 提供两种方式供您使用。

开发环境

编译并且压缩后的css、javascript文件，不含文档和源代码。

下载 YY UI

源码

yayo的所有开发源码，包括sass、JavaScript 和 字体文件、文档、工具等。

下载源码

使用

使用 YY UI 需要引入css、js文件以及字体文件。

引用css文件

            <link href="路径/yyui.css" rel="stylesheet"/>

引用js文件

yyui.js 依赖 jquery。

            <script
             src="路径/jquery.min.js"/></script>

            <script
             src="路径/yyui.min.js"/></script>

兼容性

YY UI 使用了大量的h5，css3的特性，建议运用于使用现代浏览器的场景，具体的兼容性yayo还未做测试。
