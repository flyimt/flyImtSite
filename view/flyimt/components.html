组件

Flyimt提供了一些常用的组件，可以进行快速的模块构建。

按钮组

通过按钮组容器把一组按钮放在同一行里。按钮组中的button的样式可以随意组合。

实例

将一组<button包裹在.btn-group中形成按钮组。

                <div class="btn-group">

                  <button class="btn btn-default">left</button>

                  <button class="btn btn-default">center</button>

                  <button class="btn btn-default">right</button>

                </div

尺寸

按钮组有3中尺寸可供选择，你可以在.btn-group上添加尺寸类。也可以直接在<button>上添加按钮尺寸类，这样就有4中尺寸了（多了.btn-xs）。

输入框组

通过在文本输入框<input>前面、后面或是两边加上文字或按钮，可以实现对表单控件的扩展。为.input-group赋予.input-group-addon类，可以给.form-control的前面或后面添加额外的元素。

实例

在输入框的任意一侧添加额外元素。你还可以在输入框的两侧同时添加额外元素。

$ .00 
                <div class="input-group">

                  <span class="input-group-addon">$</span>

                  <input type="text" class="form-control"/>

                  <span class="input-group-addon">.00</span>

                </div>

路径导航

在一个带有层次的导航结构中标明当前页面的位置。

各路径间的分隔符已经自动通过:before和content属性添加了。

  1. item
  2. item
  3. item
                <ol class="breadcrumb">

                  <li><a href="javascript:;">item</a></li>

                  <li><a href="javascript:;">item</a></li>

                  <li><a href="javascript:;">item</a></li>

                </ol>
为什么要用breadcrumb来作类名呢？
Because:

歪果仁们喜欢用有象征意义的名字，breadcrumb 的意思是面包屑。想想大家吃的切片面包，再看看导航路径的样式，嘿嘿嘿。。。
